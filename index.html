ToolBar

General Information

The Red Dot is the Reference Point
The Red Arrow is the Target Point
For improved precision the relationship between the Target Point and the Reference Point is controlled by the Rotation and Distance variables
Move the Reference Point using Drag and Drop
Adjust Rotation by pressing R and scrolling
Adjust Distance by pressing D and scrolling

Rotation: °



Distance: px
